'Fantasy Resort' This visual experience is programmatically created in OpenGL Fixed Function Pipeline,  just by using basic lines , triangle , quads ,  sphere , cylinder and textures used to create this demo.
              This was just a try to give a feel of tropical resort which every person dreams to go of in the midst of trees and water , Programmatically Graphical animated sequence.


Technology used : Win32 SDK , OpenGL Fixed Function Pipeline.
Operating System : Windows 10.
Programming Language : C , C++.
